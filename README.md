# OpenLandau

Translations of the sixth corrected editions of Landau-Lifshitz's seminal course in Theoretical Physics. 

Build respective chapters with:

```sh
make -C "03 - Quantum Mechanics" chapter-01
```

## Disclaimer: AI-Assisted

AI was used to transcribe, translate and typeset the documents inside this repository.
I wish I knew Russian, and had the tenacity to read it in the original, but I suppose this is the next best thing.
